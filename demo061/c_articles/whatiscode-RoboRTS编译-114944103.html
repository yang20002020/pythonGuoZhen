环境：Ubuntu18.04、opencv3.2.0

RoboRTS编译

打开终端，依次输入

mkdir -p RoboRTS_ws
cd RoboRTS_ws/src
git clone https://github.com/RoboMaster/RoboRTS
cd ..
catkin_make
